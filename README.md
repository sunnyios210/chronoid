# ChronoID

ChronoID is the first **time-evolving, soul-bound identity system built entirely on Bitcoin Layer 1**. Each token is unique, evolves automatically over time, and records reputation, achievements, and verifiable credentials directly on-chain—without any sidechains or Layer 2 solutions.

## Features
- Time-locked evolution of identity tokens
- On-chain reputation and achievements
- Privacy-preserving verification using hash commitments
- Fully Bitcoin-native using Taproot, Schnorr signatures, and CLTV
- Soul-bound, non-transferable tokens by default
